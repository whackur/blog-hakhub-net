HakHubNet은 [whackur](https://github.com/whackur)가 운영하는 IT 기술 블로그입니다.

블로그를 만든 이유는 단순합니다. 공부한 내용을 정리하지 않으면 금방 잊고, 메모만 쌓아두면 읽지 않습니다. 글로 쓰면 적어도 한 번은 제대로 정리하게 됩니다.

### 다루는 주제

**AI** — LLM의 동작 원리, 파인튜닝 실험, 에이전트 설계. 논문보다는 직접 돌려본 결과를 씁니다.

**블록체인/Web3** — Solidity 컨트랙트 작성, EVM 내부 구조, DeFi 프로토콜 분석. Ethereum 생태계를 중심으로 봅니다.

**로컬 LLM** — 클라우드 API 없이 온디바이스에서 모델을 실행하는 방법. 양자화, 추론 최적화, 프라이버시 측면을 함께 다룹니다.

**Physical AI** — 로보틱스와 엣지 AI의 접점. 센서 데이터 파이프라인, 온디바이스 추론 적용 사례를 다룹니다.

### 언어

한국어를 기본으로 쓰고, 일부 글은 영어로도 올립니다. 번역이 아니라 각 언어에 맞게 다시 씁니다.
