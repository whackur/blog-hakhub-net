HakHubNet is [whackur](https://github.com/whackur)'s personal tech blog.

The reason this blog exists is practical. Studying without writing things down means forgetting. Notes that never get organized never get read. Writing forces at least one real pass through the material.

### Topics

**AI** — LLM internals, fine-tuning experiments, agent design. The focus is on what happens when you actually run things, not just what papers claim.

**Blockchain/Web3** — Solidity contracts, EVM mechanics, DeFi protocol analysis. Mostly Ethereum-ecosystem focused.

**Local LLM** — Running models on-device without cloud APIs. Quantization, inference optimization, and the privacy angle.

**Physical AI** — Where robotics meets edge AI. Sensor data pipelines, on-device inference for real-world applications.

### Languages

Korean is the primary language. Some posts are also available in English. These are rewrites for an English-speaking reader, not literal translations.
